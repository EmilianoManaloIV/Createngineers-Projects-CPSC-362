# CPSC-362 Project
## UNOVerse Application By The Createngineers
### Introduction
The UNOVerse is a game application designed to teach the Software Development Life Cycle for CPSC-362 at California State University, Fullerton.
The application followed this process from initial requirements gathering to deployment. The following subsections will detail how to install the
game on your machine and all the subsequent documents that are attributed to this project.
### Pre-Resiquites
1. Must Use Windows 64-bit
2. Must Have A Mouse
### Installation Instructions
1. Download The UNOVerse Game.ZIP File
2. Extract The UNOVerse Game.ZIP File
3. Click On UNOVerse To Start the Game
